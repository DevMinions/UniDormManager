## 技术实现要点

### API 接口清单（完整）
```
# 换寝申请
GET    /api/room-swaps              # 获取申请列表
POST   /api/room-swaps              # 提交申请
GET    /api/room-swaps/pending      # 待审批列表
POST   /api/room-swaps/:id/approve  # 审批申请

# 查寝相关
GET    /api/inspections/rankings    # 查寝排行榜
POST   /api/inspections             # 录入评分
GET    /api/inspections/statistics  # 统计信息

# 晚归告警
GET    /api/late-returns            # 告警列表
POST   /api/late-returns/:id/handle # 处理告警

# 门禁记录
GET    /api/access-logs             # 门禁记录
GET    /api/access-logs/statistics  # 统计分析

# 楼栋管理
GET    /api/buildings               # 楼栋列表
POST   /api/buildings               # 创建楼栋

# 角色权限
GET    /api/roles                   # 角色列表
POST   /api/roles                   # 创建角色

# 消息中心
GET    /api/messages                # 消息列表
POST   /api/messages/:id/read       # 标记已读

# 数据报表
GET    /api/reports                 # 报表列表
POST   /api/reports/generate        # 生成报表

# 数据统计
GET    /api/statistics/dashboard    # 仪表盘数据
GET    /api/statistics/occupancy    # 入住统计
```
