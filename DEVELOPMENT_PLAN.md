## 技术实现要点

### API 接口清单
```
# 换寝申请
GET    /api/room-swaps              # 获取我的申请
POST   /api/room-swaps              # 提交申请
GET    /api/room-swaps/pending      # 获取待审批列表（管理员）
POST   /api/room-swaps/:id/approve  # 审批申请

# 查寝相关
GET    /api/inspections/rankings    # 查寝排行榜
POST   /api/inspections             # 录入查寝评分

# 晚归告警
GET    /api/late-returns            # 获取告警列表
POST   /api/late-returns/:id/handle # 处理告警

# 门禁记录
GET    /api/access-logs             # 获取门禁记录
```

### 页面路由规划
```
/pages/room-swaps/
  ├── list.vue      # 换寝申请列表
  ├── apply.vue     # 提交申请
  └── detail.vue    # 申请详情/审批

/pages/inspections/
  ├── rankings.vue  # 查寝排行榜（新增）
  └── score.vue     # 录入评分（新增）

/pages/late-returns/
  └── list.vue      # 晚归告警列表

/pages/access-logs/
  └── list.vue      # 门禁记录

/pages/buildings/
  └── list.vue      # 楼栋管理
```
