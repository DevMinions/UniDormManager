## 🚀 使用方式

### 微信开发者工具操作

1. **清除缓存**
   ```
   工具 → 清除缓存 → 全部清除
   ```

2. **重新导入项目**
   ```
   项目 → 删除项目
   项目 → 导入项目 → 选择 dist/build/mp-weixin
   ```

3. **开启不校验域名**（开发阶段）
   ```
   详情 → 本地设置 → 勾选「不校验合法域名」
   ```

## 📝 登录流程

```
用户打开小程序
    ↓
App.vue 检查登录状态
    ↓
├─ 已登录 → 正常显示页面
└─ 未登录 → 跳转到登录页
              ↓
         用户输入账号密码
              ↓
         登录成功 → 跳转到首页
```

## ⚠️ 注意事项

1. **首次启动**会自动检查登录状态
2. **未登录用户**只能看到登录页
3. **登录后**会自动跳转到首页
4. **底部 TabBar** 只在 TabBar 页面显示

2. 确认显示登录页

5. 点击底部 TabBar 切换页面
6. 退出登录后重新进入，确认需要重新登录
