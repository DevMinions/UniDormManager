### Key Technical Notes
- WeChat Mini Program custom TabBar MUST be at `custom-tab-bar/` in root
- File naming: `custom-tab-bar.js`, `custom-tab-bar.json`, `custom-tab-bar.wxml`, `custom-tab-bar.wxss`
- `app.json` needs `"custom": true` in tabBar config
- Component lifecycle `attached()` may not fire if path is wrong
