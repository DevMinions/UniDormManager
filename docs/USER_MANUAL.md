# UniDormManager 用户操作手册

## 📱 系统简介

宿舍管理系统是专为高校宿舍管理设计的数字化解决方案，支持学生、宿管员、维修人员、管理员等多角色使用。

---

## 👤 角色说明

| 角色 | 权限范围 | 主要功能 |
|------|----------|----------|
| 学生 | 个人相关 | 报修、申请换寝、查看评分 |
| 宿管员 | 所辖区域 | 房间管理、查寝评分、晚归处理 |
| 维修人员 | 工单处理 | 接收维修任务、更新进度 |
| 后勤管理员 | 楼栋管理 | 楼栋配置、统计报表 |
| 系统管理员 | 全部权限 | 用户管理、角色权限、系统配置 |

---

## 📲 快速开始

### 1. 登录系统

1. 打开微信小程序，搜索"宿舍管理系统"
2. 输入账号（学号/工号）和密码
3. 首次登录请修改默认密码

### 2. 首页概览

- **顶部**: 个人信息和消息通知
- **中部**: 常用功能快捷入口

## 🏠 学生操作指南

### 提交报修

1. 点击首页"我要报修"或在"报修"页面点击"新建报修"
2. 填写报修信息：
   - **标题**: 简要描述问题（如"水龙头漏水"）
   - **类型**: 选择故障类型（水电/家具/门窗等）
   - **描述**: 详细说明问题情况
   - **图片**: 上传故障照片（最多5张）
3. 点击"提交"按钮
4. 在"我的报修"中查看处理进度

### 申请换寝

1. 进入"换寝申请"页面
2. 点击"申请换寝"
3. 选择目标房间（查看可选房间列表）
4. 填写换寝原因
5. 提交申请后等待审批
6. 可在"申请记录"中查看审批状态

### 查看宿舍评分

1. 进入"宿舍评分"页面
2. 查看本月评分及历史评分趋势
3. 点击具体记录查看评分详情和改进建议

### 查看晚归记录

1. 进入"晚归记录"页面
2. 查看个人晚归历史
3. 如有疑问可联系宿管员核实

## 👨‍💼 宿管员操作指南

### 房间管理

1. **查看房间状态**
   - 进入"房间管理"页面
   - 可按楼栋、楼层筛选
   - 颜色标识：绿色（有空位）/ 红色（已满）

2. **调整学生宿舍**
   - 找到目标学生
   - 点击"调整宿舍"
   - 选择新房间并填写调整原因
   - 确认调整

### 查寝评分

1. **录入评分**
   - 进入"查寝评分"页面
   - 选择房间或扫描房间二维码
   - 按各项标准打分：
     - 卫生清洁（20分）
     - 物品摆放（20分）
     - 安全用电（20分）
     - 装饰美化（20分）
     - 文化氛围（20分）
   - 记录存在的问题
   - 上传检查照片
   - 保存评分

2. **查看评分排行**
   - 按周/月/学期查看排行
   - 导出评分报表

### 晚归处理

1. **查看晚归告警**
   - 进入"晚归告警"页面
   - 系统自动标记超过门禁时间未归的学生

2. **处理晚归记录**
   - 点击具体记录
   - 核实情况
   - 填写处理结果：
     - 正常晚归（已返校）
     - 请假在外
     - 异常情况
   - 保存处理结果

### 报修管理

1. **查看待处理报修**
   - 进入"报修管理"页面
   - 筛选"待分配"状态的工单

2. **分配维修人员**
   - 选择报修单
   - 点击"分配"
   - 选择合适的维修人员
   - 确认分配

3. **跟进维修进度**
   - 定期查看处理中工单
   - 催促超时未完成的工单

---

## 🔧 维修人员操作指南

### 接收维修任务

1. 进入"我的工单"页面
2. 查看待处理任务列表
3. 点击任务查看详情

### 处理工单

1. **前往现场**
   - 查看房间位置和问题描述
   - 携带必要工具和配件

2. **现场处理**
   - 拍照记录维修前状态
   - 进行维修处理
   - 拍照记录维修后状态

3. **完成工单**
   - 点击"完成维修"
   - 填写维修说明
   - 上传维修照片
   - 提交完成

### 查看历史工单

- 在"历史记录"中查看已完成工单
- 可按时间、类型筛选

## 🏢 后勤管理员操作指南

### 楼栋管理

1. **查看楼栋信息**
   - 进入"楼栋管理"页面
   - 查看各楼栋入住率、房间数等

2. **配置楼栋**
   - 添加新楼栋
   - 设置楼层数、每层房间数
   - 分配宿管员
   - 设置楼栋类型（男/女/混合）

3. **生成统计报表**
   - 选择报表类型（入住/报修/查寝）
   - 选择时间范围
   - 导出 Excel/PDF

### 晚归告警配置

1. 进入"系统配置"
2. 设置门禁时间
3. 配置晚归判定标准
4. 设置自动告警规则

---

## ⚙️ 系统管理员操作指南

### 用户管理

1. **添加用户**
   - 进入"用户管理"
   - 点击"新增用户"
   - 填写基本信息
   - 分配角色
   - 设置初始密码

2. **批量导入**
   - 下载导入模板
   - 按模板填写学生信息
   - 上传 Excel 文件
   - 系统批量导入

3. **权限管理**
   - 查看用户角色
   - 调整用户权限
   - 禁用/启用账号

### 角色权限配置

1. 进入"角色权限"页面
2. 查看系统预设角色
3. 可创建自定义角色
4. 为角色分配权限：
   - 页面访问权限
   - 操作权限（增删改查）
   - 数据范围权限

### 系统配置

1. **基础配置**
   - 系统名称、Logo
   - 联系方式
   - 公告设置

2. **业务配置**
   - 查寝评分标准
   - 门禁时间设置
   - 报修响应时限

3. **通知配置**
   - 微信模板消息配置
   - 短信服务配置
   - 企业微信配置

## ❓ 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码，或使用"忘记密码"功能（如已配置短信验证）

**Q: 账号被锁定？**
A: 连续输错密码5次会锁定账号，15分钟后自动解锁，或联系管理员解锁

### 功能使用

**Q: 如何修改个人信息？**
A: 进入"我的"页面，点击"个人信息"进行修改

**Q: 报修可以取消吗？**
A: 在"待处理"状态下可以取消，处理中后需联系宿管员

**Q: 换寝申请多久能审批？**
A: 一般为3个工作日内，具体看宿管员处理进度

### 技术问题

**Q: 图片上传失败？**
A: 检查网络连接，图片大小不超过10MB，格式为 jpg/png

**Q: 页面加载慢？**
A: 检查网络状况，或尝试下拉刷新重新加载
