# UniDormManager 角色化界面设计

## 🎭 系统角色体系

系统共支持 **6 种角色**，每种角色拥有专属的首页布局、功能菜单和统计数据。

### 角色列表

| 角色代码 | 角色名称 | 图标 | 颜色 | 描述 |
|----------|----------|------|------|------|
| student | 学生 | 🎓 | 蓝色 #3B82F6 | 宿舍入住学生 |
| dorm_manager | 宿管员 | 🏠 | 绿色 #059669 | 宿舍楼管理员 |
| maintenance_staff | 维修人员 | 🔧 | 琥珀色 #D97706 | 设备维修工人 |
| building_manager | 楼栋管理员 | 🏢 | 紫色 #7C3AED | 特定楼栋负责人 |
| logistics_admin | 后勤管理员 | 📋 | 红色 #DC2626 | 后勤部门管理员 |
| system_admin | 系统管理员 | ⚡ | 黑色 #1E293B | 系统超级管理员 |

## 🎨 视觉设计

### 角色色彩

每个角色拥有专属的配色方案：

```scss
// 学生 - 蓝色系
--student-primary: #3B82F6;
--student-bg: #EFF6FF;

// 宿管 - 绿色系
--dorm-primary: #059669;
--dorm-bg: #ECFDF5;

// 维修 - 琥珀色系
--maintenance-primary: #D97706;
--maintenance-bg: #FEF3C7;

// 楼栋管理 - 紫色系
--building-primary: #7C3AED;
--building-bg: #EDE9FE;

// 后勤 - 红色系
--logistics-primary: #DC2626;
--logistics-bg: #FEE2E2;

// 系统管理 - 深色系
--system-primary: #1E293B;
--system-bg: #F1F5F9;
```

### 动态背景

首页背景根据角色主色调动态变化：

```vue
<view 
  class="bg-blob" 
  :style="{ background: roleConfig.color }"
></view>
```

## 🔧 使用方法

### 1. 获取角色配置

```javascript
import { getRoleConfig, getRoleQuickMenu, getRoleStats } from '@/config/roles.js'

const role = 'student' // 用户角色
const config = getRoleConfig(role)        // 获取角色配置
const menu = getRoleQuickMenu(role)       // 获取快捷菜单
const stats = getRoleStats(role)          // 获取统计配置
```

### 2. 判断权限

```javascript
import { hasPermission, isAdmin } from '@/config/roles.js'

// 判断是否有权限
const canManage = hasPermission(userRoles, 'dorm_manager')

// 判断是否是管理员
const admin = isAdmin(userRoles)
```

### 3. 多角色支持

用户可以有多个角色，系统会：
1. 取第一个角色作为主角色显示
2. 显示"角色名等N个角色"提示
3. 使用主角色的配色方案
