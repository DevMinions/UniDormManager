## 🔐 1. 用户认证模块

### 1.1 用户登录
- **功能描述**: 用户使用用户名密码登录系统
- **输入**: username, password
- **预期输出**: JWT token + 用户信息
- **测试步骤**:
  1. 访问登录页面
  2. 输入正确用户名密码
  3. 验证token返回
  4. 验证页面跳转
- **边界情况**:
  - 错误密码 → 返回401
  - 空用户名 → 返回400
  - 不存在的用户 → 返回404

### 1.2 微信登录
- **功能描述**: 微信小程序登录
- **输入**: wxCode, userInfo
- **预期输出**: JWT token
- **状态**: ⏳ 待测试

### 1.3 获取当前用户
- **功能描述**: 获取当前登录用户信息
- **接口**: GET /api/auth/me
- **预期输出**: 用户详情
- **状态**: ⏳ 待测试

### 1.4 用户登出
- **功能描述**: 退出登录
- **接口**: POST /api/auth/logout
- **预期输出**: 成功消息
- **状态**: ⏳ 待测试
